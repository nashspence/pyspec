Create a minimal but strongly validated contract-first starter for PM/design agents. The primary source is a sparse direct authored contract (`contract.yaml`) that compiles into `generated/contract.complete.yaml`, OpenAPI, AsyncAPI, CWL, routing, panel contracts, one generated Gherkin corpus for both pytest-bdd harnesses, and full-contract visual audit renders. Patch operations remain available as a constrained-agent edit format that can materialize or refresh the authored contract. The PM/design contract must not contain test-harness routing, dev-environment details, SQL, database dialects, migrations, or ORM/storage implementation details.
